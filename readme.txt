Custom Extension for WooCommerce to show the product short description in the WooCommerce product archives


== Description ==

Shows short descriptions for each product on the WooCommerce product archives to give more detail about each product for your customers


== Frequently Asked Questions ==

= What does this plugin do? =

Adds the product short description to each product on the Product Archives page. Typically this is under the product featured image, however the position is dependant on how WooCommerce is configured.

= The product archives are not showing descriptions =

You are required to set the product short description for each product. You can find this field towards the bottom of the edit product page.
